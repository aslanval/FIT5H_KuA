	>>		dito, aber das Ergebnis wird angehängt

			ls -l /etc | less
				Die Anzeige der Dateien/Ordner vom Verzeichnis etc wird an das Programm less übergeben, dass eine seitenweise ANzeige von Text berherrscht

Links
	Hardlink
		ln <dateiname> <dateiname-lnk>
			erzeugt eine neue Referenz auf eine Datei in der File-Tabelle der Festplatte

	Softlink
		ln -s <verzeichnisname> <verzeichnisname-neu>
			Da Hardlinks fur für Dateien möglich sind, gibt es den Softlink für Verzeichnisse und Dateien.
			Es wird eine neue Datei erstellt, deren Inhalt der verweis auf das Verzeichnis ist.
